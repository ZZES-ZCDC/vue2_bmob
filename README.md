# 字段

 - FEvaluate	报修报价
 - FHandlePerson	工单处理人员
 - FHandleResults	处理结果
 - FId	用户tokenID
 - FName	报修名称
 - FSerialNumber	报修序列号
 - FServiceDate	报修日期
 - FStatus	处理进度
 - wilsonsocre	其他

# api请求方式
请看`postman`文件夹，导入到postman软件中测试

# 启动方式
```
$ git clone https://github.com/ZZES-ZCDC/vue2_bmob.git
$ cd vue2_bmob
$ npm install
$ npm run dev
```
